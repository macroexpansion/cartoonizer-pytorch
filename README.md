## Cartoonizer Pytorch
This Pytorch implementation contains only inference code based on Tensorflow implementation from [White Box Cartoonization](https://github.com/SystemErrorWang/White-box-Cartoonization)

## Requirements
- python >= 3.6
- pytorch >= 1.5
- opencv-python >= 4.3
- numpy >= 1.18
